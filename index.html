PezzaliDashboard

Dashboard analitica per il tuo profilo GitHub. I tuoi dati restano nel tuo browser, nessun server intermedio, nessun account: solo tu, il tuo token, le tue metriche.

Token classico o fine-grained, con permessi minimi: public_repo (per repo pubblici) e read:user. Come si crea?
Come creare un Personal Access Token:
  1. Vai su github.com/settings/tokens
  2. Clicca Generate new token → Classic
  3. Note: PezzaliDashboard · Scadenza: a tua scelta
  4. Spunta: public_repo, read:user, read:org
  5. Clicca Generate token, copia il token (inizia con ghp_)
  6. Incollalo qui sopra e accedi
🛡️ Il token viene salvato solo in localStorage del tuo browser, non lascia mai il tuo dispositivo. Nessun server di terze parti, nessun tracking, nessun account. Puoi revocarlo in qualsiasi momento da GitHub.

🌱 Open source · Federato · Per ognuno il suo

PezzaliDashboard è gratuita, open source e forkabile. Vuoi avere la tua istanza privata, con il tuo brand, sul tuo dominio? Fai un fork del repo, configurala con il tuo token, e diventa la tua personale console di GitHub analytics.
